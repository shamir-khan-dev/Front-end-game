# 🐉 Dragon Repeller – JavaScript RPG Game

Dragon Repeller is a fully playable, browser-based RPG game built using **HTML, CSS, and vanilla JavaScript**.  
This project was created as part of my frontend learning journey to solidify core web development concepts through hands-on practice.

## 🎮 Game Features
- Player stats system (XP, Health, Gold)
- Multiple locations (Town Square, Store, Cave)
- Turn-based combat system (Attack, Dodge, Run)
- Multiple enemies (Slime, Lion, Dragon)
- Weapon and inventory system
- In-game shop (buy health & weapons)
- Randomized damage and outcomes
- Easter egg mini-game
- Win, lose, and restart conditions

## 🧠 What I Learned
- HTML structure and semantic elements
- CSS styling and layout fundamentals
- JavaScript fundamentals (variables, functions, arrays, objects)
- DOM manipulation and event handling
- Game state management
- Conditional logic and loops
- Debugging and problem-solving in JavaScript
